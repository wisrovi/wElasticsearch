Glossary
========

.. glossary::
   :sorted:

   Elasticsearch
      Distributed search and analytics engine built on Apache Lucene.

   Index
      Collection of documents with similar characteristics, analogous to a database.

   Document
      Basic unit of information in Elasticsearch, stored as JSON.

   Mapping
      Schema definition for documents in an index.

   Shard
      Horizontal partition of an index, enabling distributed storage and search.

   Query
      Request to search and filter documents in an index.

   Aggregation
      Framework for computing aggregated data from documents.

   Bulk Operations
      Multiple operations executed in a single request for efficiency.

   Pydantic
      Python data validation library used for document modeling.

   ORM
      Object-Relational Mapping. Technique for mapping data between Python objects and databases.
